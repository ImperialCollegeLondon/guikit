Initialises repository with the files/folders required to use pyguitemp.

This is done by copying the skeleton repository structure within pyguitemp to the desired location.

Parameters

  • path (pathlib.Path) – Where the repository should be initialized. Default: current.

  • name (str) – Name of the application.
